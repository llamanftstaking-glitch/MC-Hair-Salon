  On Replit — 3 steps before going live                                                                                                                                                                                                     
                                                                                                                                                                                                                                            
  1. Add the DATABASE_URL secret                                                                                                                                                                                                            
  - Go to Replit → Secrets tab                                                                                                                                                                                                              
  - Add DATABASE_URL → use the connection string from your Replit PostgreSQL add-on                                                                                                                                                         
  - Also add SALON_INBOX_EMAIL=hello@mchairsalon.com if not already there                                                                                                                                                                   
                                                                                                                                                                                                                                            
  2. Create the database tables                                                                                                                                                                                                             
  npm run db:push                                                                                                                                                                                                                           
                                                                                                                                                                                                                                            
  3. Import your existing data (one-time, run once)                                                                                                                                                                                         
  node scripts/import-data.js                                                                                                                                                                                                               
  This moves your existing admin account, staff, and settings from the JSON files into Postgres.                                                                                                                                            
                                                                                                                                                                                                                                            
  4. Deploy                                                                                                                                                                                                                                 
  npm start will now seed the admin into the DB automatically on every boot.                                                                                                                                                                
                                                                                                                                                                                                                                            
  ---                                                                                                                                                                                                                                       
  After this, everything — customer accounts, bookings, gift cards, packages, messages — survives forever across redeploys. You're launch-ready.
